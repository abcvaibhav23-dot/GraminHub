GraminHub क्या है?

GraminHub गांव-कस्बों के लिए सप्लायर और वाहन सेवाओं का आसान प्लेटफॉर्म है। आप जरूरत लिखें, सप्लायर देखें, और Call/WhatsApp से तुरंत संपर्क करें।

बड़े बटन, आसान स्क्रीन

सीधा Call/WhatsApp

बुकिंग स्टेटस ट्रैकिंग

Buyer/User कैसे इस्तेमाल करे?

  1. 1) Search box में अपनी जरूरत लिखें (जैसे: सीमेंट, ट्रक, ट्रैक्टर)।
  2. 2) नीचे category-wise items में से सप्लायर चुनें।
  3. 3) “Call” या “WhatsApp” दबाकर सीधे बात करें।
  4. 4) बुकिंग ट्रैक करने के लिए OTP से लॉगिन करें।

Tip: अपनी जगह (गांव/शहर) और मात्रा जरूर लिखें।

Supplier कैसे इस्तेमाल करे?

  1. 1) OTP से Supplier के रूप में लॉगिन करें।
  2. 2) प्रोफाइल बनाएं (नाम, फोन, पता)।
  3. 3) अपनी items/services और price जोड़ें।
  4. 4) Owner/Admin verification के बाद आपकी लिस्टिंग live होगी।

Tip: सही फोन नंबर + साफ पता भरोसा बढ़ाता है।

लॉगिन से पहले क्या कर सकते हैं?

  • Items और suppliers category-wise देखें।
  • Search (type/voice) से supplier ढूंढें।
  • Call/WhatsApp करके तुरंत बात शुरू करें।

लॉगिन के बाद क्या extra मिलेगा?

  • अपनी बुकिंग status ट्रैक करें।
  • Supplier के लिए: प्रोफाइल + items/services manage करें।
  • Supplier verification और booking requests dashboard में देखें।

Step-by-step Use Case

OTP login में अलग से register जरूरी नहीं है — पहली बार OTP लेने पर account अपने-आप बन जाता है।

Buyer/User

  1. 1) Home पर search box में अपनी जरूरत लिखें या voice से बोलें।
  2. 2) नीचे category-wise items में supplier card देखें।
  3. 3) Call/WhatsApp से बात करके price + delivery confirm करें।
  4. 4) Booking status track करने के लिए OTP से login करें (no register needed)।

Supplier

  1. 1) OTP से Supplier के रूप में login करें।
  2. 2) Supplier Dashboard में प्रोफाइल भरें (business name, phone, address)।
  3. 3) Items/services जोड़ें: item name + unit type (per trip/per day/per bag) + price.
  4. 4) Owner/Admin verification के बाद आपकी listing users को दिखेगी।

Owner/Admin

  1. 1) Allowlisted phone से OTP login करें (Owner/Admin)।
  2. 2) Admin Dashboard में pending suppliers approve/reject करें।
  3. 3) Fraud control: block/unblock, reviews/complaints देखें।
  4. 4) Owner settings (Call/WhatsApp enable/disable) manage करें।

सुरक्षा/टिप्स

  • फोन पर कीमत, डिलीवरी समय, और कुल मात्रा पहले confirm करें।
  • Unknown लिंक/OTP किसी को न दें।
  • समस्या हो तो ऊपर “Help: Call/WhatsApp” से संपर्क करें।

Owner/Admin (Production)

Production में Admin role सिर्फ allowlisted नंबरों के लिए है। Admin का काम ऑपरेशन और सुरक्षा है (रोज़मर्रा buyer/supplier use नहीं)।

  • Supplier verification/review: documents देखकर verify/reject करें।
  • Suspicious suppliers/users को block/suspend करें।
  • WhatsApp/Call enable/disable जैसी owner settings नियंत्रित करें।

Note: Production में `OTP_EXPOSE_IN_RESPONSE=0` रखें और OTP delivery provider (SMS/WhatsApp) configure करें।
